Using the interface

Main Interface

The main interface consists of a header and three center panels, the menu panel, the content panel and the help panel (this panel).

Header

On the top of the page we have the heather. Inside it you can see the application's logo on the left, the user box on the right and, below these two, a bar containing the crumbs on the left and a search box on the right.

The user box tells you which user you are logged in as. If you ever see here a username other than yours you should logout and log in back with your own username. Also on the user box you can Logout, view or edit your Account info and show or hide this Help.

The crumbs tell you which is your active project and where are you located inside the application. The active project is the project you are currently working on, and is where new files, messages, etc. are stored when you create them. You can change the active project on the menu panel.

Finally, the search box lets you perform a search of objects in the active project.

Menu Panel (1/3)

The menu panel contains links to the different sections of the application, grouped in four categories: Overview, Documents, Projects and Administration (administrators only).

This panel can be resized by dragging the right edge. To do this place the mouse over the right edge of the menu panel, hold down the left mouse button and move left or right; then release the left button. You can also hide it completely by clicking the « button on the top right, and you can bring it back in by pressing the » button.

Menu Panel (2/3)

The Overview category provides the following links:

  • Dashboard, where you get to see a summary of the recent activities that concern you;
  • Account, where you can modify your account information;
  • Help, to show or hide this help;
  • Logout, to logout of the application;

The Documents category lets you browse all files related to you. You can filter by project, by tags or by type or just list the files created by you. You also have a shortcut to the last three modified files for quicker access to your recent work, and shortcuts to add files.

Menu Panel (3/3)

The Projects category lets you manage the projects of which you are a member. First you should select the active project, which is the project you will be working on. You can do this by selecting the project on the combobox (expansible list) below Select project:. Then, you can add messages, tasks, files, etc., or just add a new project with the add project action.

Finally, only if you are an administrator, you can see the Administration category. Here you can configure the application, edit the company's info, manage users and clients, and more.

Content Panel

This panel is the heart of the application. This is where the different sections of OpenGoo are displayed. To learn more about teh contents of this panel check the other sections in Help or go to OpenGoo's site at http://www.opengoo.org and check the forum and wiki.

Help Panel

This panel. You can resize this panel by dragging its left edge. To do this place the mouse over the left edge of the help panel, hold down the left mouse button and move left or right; then release the left button. You can also hide it completely by clicking the » button on the top right, and you can bring it back in by pressing the « button or the other Help buttons available on the user box and on the menu.
