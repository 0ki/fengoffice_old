Using the interface

Main Interface

The main interface consists of a header and four center panels, the workspace panel, the tag panel, the content panel and the help panel (this panel).

Header

On the top of the page we have the heather. Inside it you can see the application's logo on the left, followed by the search box and the user box on the right.

The user box tells you which user you are logged in as. If you ever see here a username other than yours you should logout and log in back with your own username. Also on the user box you can Logout, view or edit your Account info, perform Administration tasks (if you are an administrator) and show or hide this Help.

The search box lets you perform a search of objects in the active project.

Workspace Panel

The workspace panel contains a list of your active workspaces. Here you can add a new workspace or rename and delete an existing one, if you have the appropiate permissions.

By selecting one of the workspaces you affect the content of the content panel. For example, only the documents of the selected workspace are shown on the Document's tab.

Tag Panel

The Tag panel lists the tags relevant to the selected workspace. By selecting a tag from this list you can filter the contents of the content panel (if applicable)

Content Panel

The content panel is divided into seven tabs, which are:

  • Overview
  • Messages
  • Contacts
  • Calendar
  • Documents
  • Tasks
  • Web pages

Overview

This tab lists all of the Content Objects of the selected workspace which are tagged with the selected tag. This panel is useful, when sorting the objects by the last update date, to get a glimpse of the recent activities in the selected workspace.

Messages

Here you can access messages of the selected workspace.

Help Panel

This panel. You can resize this panel by dragging its left edge. To do this place the mouse over the left edge of the help panel, hold down the left mouse button and move left or right; then release the left button. You can also hide it completely by clicking the » button on the top right, and you can bring it back in by pressing the « button or the other Help buttons available on the user box and on the menu.
